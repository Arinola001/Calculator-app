Calculator App

A responsive calculator built with HTML, CSS, and JavaScript. It performs basic arithmetic operations and features a clean, modern interface optimized for all devices.

 Features
- Addition, subtraction, multiplication, and division
- Decimal support
- All Clear (AC) functionality
- Responsive design

Technologies
- HTML5
- CSS3
- Vanilla JavaScript
